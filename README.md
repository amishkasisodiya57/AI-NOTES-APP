# 🧠 AI Notes App

A simple and clean AI-powered Notes App built using React.

## 🚀 Features
- Create Notes
- Delete Notes
- Fetch data from API
- AI-based text summarization
- Clean and modern UI

## 🛠️ Tech Stack
- React (Vite)
- JavaScript
- HTML/CSS
- JSONPlaceholder API

## 📦 How to Run
npm install  
npm run dev  

## 💡 Description
This app allows users to create and manage notes. It also includes a basic AI feature that summarizes long text into short content.
